\# 🎵 J.A.R.V.I.S Music Player (Python)



A lightweight Python-based background music player inspired by \*\*JARVIS\*\*, capable of playing songs directly from the internet (YouTube audio) without opening any browser or video window.



\---



\## 🚀 Features



\* 🎧 Play any song by typing name

\* 🔇 No browser / no video (audio only)

\* ⚡ Fast YouTube audio streaming

\* 🔁 Change songs anytime

\* 🖥 Runs inside terminal (VS Code supported)

\* 💯 Free and open-source



\---



\## 🛠 Requirements



Make sure you have the following installed:



\* Python 3.10+

\* VLC Media Player

\* Internet connection



\---



\## 📦 Installation Steps



\### 1️⃣ Clone or Download Project



```bash

git clone <your-repo-link>

cd <project-folder>

```



\---



\### 2️⃣ Create Virtual Environment (Recommended)



```bash

python -m venv venv

venv\\Scripts\\activate

```



\---



\### 3️⃣ Install Dependencies



```bash

pip install yt-dlp python-vlc

```



\---



\### 4️⃣ Install VLC Media Player



Download and install VLC from official website:



https://www.videolan.org/vlc/



Default installation path should be:



```

C:\\Program Files\\VideoLAN\\VLC

```



\---



\### 5️⃣ Run the Program



```bash

python J\_A\_R\_V\_I\_S.py

```



\---



\## 🎧 How to Use



\* Enter song name when prompted:



```

Enter song name: kesariya

Enter song name: arijit singh songs

```



\* Press:



```

Ctrl + C

```



👉 To stop current song and enter a new one



\* Type:



```

exit

```



👉 To close the program



\---



\## ⚠️ Troubleshooting



\### ❌ No Sound Issue



\* Check system volume

\* Ensure correct audio output device

\* Test audio in VLC manually



\### ❌ VLC Error (libvlc.dll not found)



\* Make sure VLC is installed

\* Verify path:



&#x20; ```

&#x20; C:\\Program Files\\VideoLAN\\VLC

&#x20; ```
